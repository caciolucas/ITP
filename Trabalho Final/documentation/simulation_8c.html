Faz a simulação acontecer usando as demais funções.

Aloca espaço pros 3 cenários usando allocCenario calcula as taxas b,k e b2,k2 usando calcB e calcK com os parâmetros recebidos no arquivo de entrada. Simula cada um dos cenários com a simulateCenario e então, escreve cada um dos cenários no arquivo de saida csv com writeCenarios.

Parâmetros
tQuantidade de iterações que a simulação terá
outputPonteiro pro arquivo csv onde serão escritos os dados simulados
outputFileNameNome do arquivo csv de saída usado para print no terminal
